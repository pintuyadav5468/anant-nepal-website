# Anant Nepal Website

Welcome to the Anant Nepal website project! This project aims to create a modern and dynamic web application for Anant Nepal, inspired by the design and functionality of ryc.org.np. The website will showcase the organization's mission, programs, and impact, while providing a user-friendly experience.

## Project Structure

The project is organized as follows:

```
anant-nepal-website
├── public
│   ├── index.html        # Main HTML document
│   └── logo.jpg          # Logo image for branding
├── src
│   ├── components
│   │   ├── Logo.jsx      # Logo component
│   │   └── ProgramCard.jsx # Program card component
│   ├── pages
│   │   ├── Home.jsx      # Homepage component
│   │   └── Programs.jsx   # Programs listing component
│   ├── App.jsx           # Main application component
│   ├── index.js          # Entry point for the React application
│   └── styles.css        # CSS styles for the website
├── package.json          # npm configuration file
└── README.md             # Project documentation
```

## Features

- **Responsive Design**: The website is designed to be fully responsive, ensuring a seamless experience across devices.
- **Dynamic Content**: The use of React allows for dynamic rendering of content, making it easy to update and manage.
- **User Interaction Logging**: The application logs user interactions for better understanding of user engagement.
- **Future-Ready**: The project is built with scalability in mind, allowing for future enhancements and features.

## Setup Instructions

To get started with the Anant Nepal website project, follow these steps:

1. **Clone the Repository**:
   ```
   git clone <repository-url>
   cd anant-nepal-website
   ```

2. **Install Dependencies**:
   ```
   npm install
   ```

3. **Run the Application**:
   ```
   npm start
   ```

4. **Open in Browser**:
   Navigate to `http://localhost:3000` to view the application.

## Usage Guidelines

- The homepage provides an overview of Anant Nepal's mission and impact.
- The programs page lists all available programs with filtering options.
- Users can interact with the logo and program cards to log their engagement.
